# Mini shopping-mall
## 기술

HTML, Vanilla JS, CSS 연습용 웹사이트

## 기능

- 아이템 스크롤링
- data 동적으로 받아오기
- 버튼을 이용한 간단한 필터링 기능
- 상점 아이콘 클릭시 데이터를 전부 가져옴

## 기간

2021.07.14~2021.07.15
